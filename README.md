# CSC462_RAFTLAB

RAFT code in Golang for CSC462 Summer 2020. Base code provided. Modified code was completed under src\kvraft, src\raft, and src\shardkv. 

Build code under folder kvraft with: 

```
go build
```

Then run the code with:

```
go test
```
